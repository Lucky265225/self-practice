เรื่อง Array
มีการเก็บข้อมูลเป็นลำดับ โดยแต่ละตัวจะมี index ในการเข้าถึง ซึ่งตัวมันจะไม่ได้เก็บข้อมูลไว้ที่ตัวเองจะมีการอ้างถึงไปยังแหล่งที่เก็บ
ซึ่ง array ก็สามารถ เก็บค่าได้ทั้ง string number หรือ จะเป็น array object ก็ได้
array ใน javascript จะเริ่มจากตำแหน่ง ที่ 0

จากการทำโจทย์
- ได้ลองเจ้าถึง array มากขึ้น ผ่าน การใช้ forloop
- การบวกค่าใน array
- การหาค่าที่ต้องการโดยการ loop แล้วใช้ if ในการช่วยเช็คเงื่อนไข
- การแยกโครงสร้าง ของกลุ่ม array ออกมาเป็น หลายๆก้อนด้วย ... destructuring
- การรวมค่าหลายค่าจาก argument มาเป็น array ด้วย ... rest parameter
