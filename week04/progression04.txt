เรื่อง higher-order function

map
- ใช้ในการแปลงข้อมูลแต่ละตัวใน array แล้วนำออกมาเป็น array ตัวใหม่

filter
- ใช้ในการกรองข้อมูลใน array ที่เราต้องการตามเงื่อนไข แล้วนำออกมาเป็น array ตัวใหม่

reduce
- ใช้ในการรวมค่าใน array ให้เหลือตัวเดียว ไม่ว่าจะเป็น การดำเนินการทางคณิตศาสตร์ หรือการต่อกันแบบ concat

จากการทำโจทย์
- ได้เรียนรู้วิธีการแปลงค่าได้รวดเร็วและกระชับมากขึ้น
- สามารถดึงข้อมูลที่ต้องการใช้มาเป็น array ใหม่ได้เร็วขึ้น
- ฝึกการคิดที่ลดขั้นตอนการเขียนโค้ดที่ซับซ้อน
- ฝึกการกรองข้อมูลที่ต้องการก่อน แล้วนำข้อมูลนั้นมาใช้ต่อได้เลย
