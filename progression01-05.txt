จากเรื่องที่เรียนมาทั้งหมด 5 สัปดาห์ ทำให้เข้าใจเนื้อหามากยิ่งขึ้น การได้ลองใช้ function ต่างๆ
ได้ลองทำโจทย์ ทำให้รู้ว่า ปัญหาแบบไหนควรใช้วิธีการไหนแก้ ทำให้แก้ปัญหาได้ตรงจุดและเร็วขึ้น
เรื่อง
- array รู้วิธีการเข้าถึง การดึงข้อมูลมากขึ้น
- object ได้ฝึกวิธีการเขียน การเก็บแบบแค่ตัวเอง หรือเป็นแม่แบบให้กับตัวอื่นๆด้วย
- function ฝึกการเขียนให้เป็นระบบและเรียกใช้ซ้ำได้ดีขึ้น
- higher-order ฝึกการเขียนรูปแบบ callback ทำให้จัดการได้ง่ายมากขึ้น และเขียนได้สั้นกกระชับในการแก้ปัญหา
- export ข้อมูลที่ต้องการนำไปใช้กับไฟล์อื่น
- import นำข้อมูลที่ต้องการใช้จากไฟล์อื่นเข้ามา
